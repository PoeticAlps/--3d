## 🔄 完整工作流

### 阶段一：素材准备 ✅
```
1. 收集圆明园历史图片资料
   ↓
2. 参考《圆明园四十景图》等古画
   ↓
3. 搜索现代复原图、考古资料
   ↓
4. 整理到 data/raw/images/ 目录
```

### 阶段二：AI 3D重建
```
1. 选择核心建筑（如：正大光明、九州清晏）
   ↓
2. 使用 TripoSR / Meshy API 图片转3D
   ↓
3. 输出基础模型 (.glb/.obj)
   ↓
4. AI纹理修复与增强
   ↓
5. 模型质量检查与优化
```

### 阶段三：Blender场景组装
```
1. 导入AI生成的基础模型
   ↓
2. 修复模型拓扑与法线
   ↓
3. 添加材质、纹理、光照
   ↓
4. 组装完整场景（山水、建筑、植被）
   ↓
5. 设置摄像机与渲染参数
   ↓
6. 导出最终模型 (GLB格式)
```

### 阶段四：Web3D展示开发
```
1. Three.js场景搭建
   ↓
2. 模型加载与优化
   ↓
3. 交互功能开发：
   - 鼠标控制视角
   - 滚轮缩放
   - 点击查看详情
   - 新旧对比时间轴
   ↓
4. UI界面设计
   ↓
5. 性能优化与测试
```

### 阶段五：作品包装
```
1. 制作宣传海报
   ↓
2. 编写设计说明文档（200字以内）
   ↓
3. 制作演示视频（可选）
   ↓
4. 编写制作报告
   ↓
5. 整理AIGC使用声明
```

### 阶段六：提交
```
1. 作品打包
   ↓
2. 上传至3D大赛小程序/网站
   ↓
3. 等待学校审核推荐
   ↓
4. 完成提交
```
