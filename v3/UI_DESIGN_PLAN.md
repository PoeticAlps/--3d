# 圆明园3D重建 · UI设计升级计划

## 当前状态（V3 baseline）

| 模块 | 现状 | 问题 |
|------|------|------|
| Header | 金色标题 + 比赛徽章 | 静态，无动效 |
| 景区导航 | 右侧5个按钮 | 无图标，纯文字 |
| 数据面板 | 左上角4项数据 | 数字静态，无计数动画 |
| 详情弹窗 | 左下角描述框 | 弹出无过渡 |
| 操作提示 | 底部居中固定文字 | 无手势图标 |
| Loading | 进度条 + 标题 | 过于简单 |
| 总体 | 暗色+金色玻璃态 | 缺响应式、缺微交互、缺沉浸感 |

## 设计系统（ui-ux-pro-max 生成）

### 色彩
- **主色**: `#0D0F1A`（深邃夜空蓝）
- **金 accent**: `#D4AF4B` → `#FFD700` 渐变
- **辅助**: `#1A1F35`（面板底）、`#2A2F45`（hover）
- **文字**: `#F0E6D3`（主文字）、`#8B8370`（次要文字）
- **危险/警告**: `#C44536`（红色石柱）

### 字体
- **标题**: `Ma Shan Zheng`（毛笔书法感）
- **副标题**: `ZCOOL XiaoWei`（文雅）
- **正文**: `Noto Serif SC`（宋体）
- **数据**: `ZCOOL QingKe HuangYou`（像素感数字）

### 圆角 & 阴影
- 面板: `12px` 圆角 + `0 8px 32px rgba(0,0,0,0.6)` 阴影
- 按钮: `8px` 圆角
- 标签: `10px` 圆角
- 边框: `1px solid rgba(212,175,75,0.15)` 金色微光

## 升级计划（6个Phase）

### Phase 1: Loading沉浸感（最高优先）
- **目标**: 进入瞬间抓住用户
- **改动**:
  - [ ] 标题文字 `fadeUp` 动画（0→1 + 上移20px，0.8s ease）
  - [ ] 进度条改为金色脉冲光效
  - [ ] 加载完成后 `scale(1.05)` → `scale(1)` 缩放消失
  - [ ] 背景加微粒（Canvas粒子，数量50，随机飘动）

### Phase 2: Header动态化
- **目标**: 品牌感 + 竞赛信息
- **改动**:
  - [ ] 标题 `letter-spacing` 从 `4px` 渐变到 `2px`（打字机效果）
  - [ ] 竞赛徽章改为可折叠下拉（显示年份/赛道/类别）
  - [ ] 添加「全屏」按钮（右上角，`⛶` 图标）
  - [ ] Header 添加 `backdrop-filter: blur(12px)` 毛玻璃效果

### Phase 3: 景区导航重设计
- **目标**: 直观、有层级、有反馈
- **改动**:
  - [ ] 每个按钮添加 SVG 图标（建筑剪影/树/水/宫殿）
  - [ ] hover 时 `translateX(4px)` + 左边框金色竖线
  - [ ] active 状态添加 `box-shadow: inset 0 0 12px rgba(212,175,75,0.15)`
  - [ ] 按钮间添加分隔线（`1px solid rgba(255,215,0,0.08)`）
  - [ ] 添加「收起」功能（侧边栏可折叠为图标条）
  - [ ] 切换景区时添加淡入淡出过渡（`opacity 0.3s`）

### Phase 4: 数据面板升级
- **目标**: 数据可视化的仪式感
- **改动**:
  - [ ] 数字 `countUp` 动画（从0到目标值，1.5s easeOut）
  - [ ] 每项数据添加小图标（🏛️建筑/📐面积/📚文献/🧊模型）
  - [ ] 进度条可视化（每个数据项下方一条细进度条）
  - [ ] 面板可折叠（点击标题折叠/展开）
  - [ ] 添加「AI分析」状态指示（闪烁绿点 + "分析中..."文字）

### Phase 5: 详情面板增强
- **目标**: 沉浸式信息展示
- **改动**:
  - [ ] 弹出动画: `translateY(20px)` → `translateY(0)` + `opacity 0→1`
  - [ ] 添加景点历史时间线（竖向时间轴，关键年份节点）
  - [ ] 添加「复原前/后」对比滑块（两张图左右拖动对比）
  - [ ] 关闭按钮改为右上角 `×` + 快捷键 `Esc`
  - [ ] 标签改为可点击（点击筛选同类型景点）

### Phase 6: 沉浸体验（高级）
- **目标**: 从"查看器"升级为"体验"
- **改动**:
  - [ ] 添加环境音效开关（右上角🔊按钮）
  - [ ] 小地图（右下角，俯瞰缩略图 + 摄像机位置标记）
  - [ ] 分享按钮（生成截图 + 复制链接）
  - [ ] 键盘快捷键 `1-5` 直接切换景区
  - [ ] 鼠标移到景区建筑上时高亮 + 浮动名称标签

## 执行顺序

```
Phase 1 → Phase 2 → Phase 3 → Phase 4 → Phase 5 → Phase 6
  ↑            ↑          ↑          ↑          ↑          ↑
Loading      Header    导航栏     数据面板   详情面板    高级体验
(必做)       (必做)     (必做)     (推荐)     (推荐)     (可选)
```

**预计工时**: Phase 1-3 约1小时（核心），Phase 4-6 可分批做

## 技术约束

- 单HTML文件，内嵌CSS/JS（不拆分）
- Three.js r150 + 内联OrbitControls
- 不依赖外部CDN（网络不通）
- 保持现有3D场景不变，只改UI层
- 所有动画用CSS transitions/animations，避免JS动画库
